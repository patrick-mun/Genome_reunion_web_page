# Genome_reunion_web_page

Page web de communication pour le projet **Génome Réunion**.

## Objectif

Présenter le projet à un public non spécialiste, avec un ton accessible mais compatible avec une communication institutionnelle et scientifique.

## Dernières corrections

- Stabilisation du vocabulaire autour de **référentiel génomique réunionnais** plutôt que "génome de référence" au sens strict.
- Remplacement de l'expression "4 IA" par **modules d'aide à l'analyse clinique**, moins ambiguë pour un public institutionnel.
- Reformulation des bénéfices cliniques pour éviter les promesses trop directes ou non encore validées.
- Suppression des dates de collecte fictives et des formulations laissant croire à un lancement opérationnel déjà validé.
- Ajout de mentions de prudence sur les conventions, autorisations, documents d'information et consentement.
- Correction du JavaScript commun afin qu'il fonctionne aussi sur les pages sans section `#hero`.

## Fichiers principaux

- `index.html` : page d'accueil du projet.
- `participer.html` : page d'information sur les futures modalités de participation.
- `styles.css` : charte graphique et styles communs.
- `script.js` : animations, barre de progression et comportements communs.

## Point d'attention

Avant diffusion officielle, vérifier avec l'équipe projet :

- la liste définitive des partenaires ;
- le statut des conventions EFS/CRB/CHU/Université/POPgen ;
- l'adresse de contact officielle ;
- les formulations validées par les instances éthiques, RGPD et communication institutionnelle.
